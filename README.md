Descripción
PlanAR es una plataforma web interactiva orientada a la comunidad para el descubrimiento de eventos localizados como recitales, ferias gastronómicas y actividades culturales independientes. Este proyecto simula un flujo de trabajo profesional ágil utilizando control de versiones y control de calidad (QA).

 Tecnologías Utilizadas
 Frontend: HTML, CSS, JavaScript 
Control de Versiones: Git & GitHub

Funcionalidades Desarrolladas
1.  Módulo de Eventos: Creacion adaptativa mediante tarjetas informativas que reaccionan al pasar el cursor (efecto hover).
2.  Reservas Dinámicas: Confirmación en tiempo real en la interfaz del usuario al clickear en "Reservar lugar", prescindiendo de ventanas emergentes invasivas.
3.  Inyección de Categorías: Renderizado dinámico de tópicos mediante manipulación del DOM utilizando arreglos nativos de JavaScript.
